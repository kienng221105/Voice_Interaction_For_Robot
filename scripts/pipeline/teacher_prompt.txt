- buy_product
- add_product
- change_product
- payment
- cancel
- show_menu
- greeting
- help
- unknown

Valid Products (Catalog):
coca, pepsi, sting, aquafina, 7up

1. Analyze the user's input text. Note that the text may contain ASR (speech-to-text) errors (e.g. "a qua phi na" -> "aquafina", "xtinh" -> "sting", "bAcp si" -> "pepsi").

3. Extract the products mentioned and map them to the Valid Products catalog.
4. Extract the exact quantity explicitly mentioned for each product. If no quantity is mentioned, default to 1 or omit.
5. Provide your response strictly as a JSON object matching the schema below.

Output JSON Schema:
{
  "intent": "string",
  "entities": {
    "items": [
      {
        "product": "string",
        "quantity": number
      }
    ]
  }
}

Example:
Input: "cho mình hai ly xtinh với một pep xi"
Output:
{
  "intent": "buy_product",
  "entities": {

      {
